# BobuxAdmin
Source code of BobuxAdmin bot from Bobux Bot Development server. The bot is written with usage of `disnake` and SQLite database.

## Functionality
Provides basic moderation features - mute, ban, kick and purge commands. Also has a system of bug catcher points.

## Usage
You can use this code for personal purpose, but make sure to encrypt your token with `token_encryptor.py` and change the values in `constants.py` if you are copying the full code, otherwise nothing will work.
